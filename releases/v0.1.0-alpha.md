# SensorGenome v0.1.0-alpha

**Title:** SensorGenome v0.1.0-alpha: Initial open foundation for AI-driven molecular sensing

## Summary

This alpha release establishes the first public scaffold for SensorGenome: an open foundation for AI-driven molecular sensing, probe-response benchmarks, uncertainty-aware baselines, and active-learning workflows.

## Included

- Python package scaffold
- 25-row seed probe-response dataset
- JSON schema for probe-response records
- CLI for validating datasets
- CLI for training a baseline response-ratio model
- CLI for selecting next experiments with simple acquisition logic
- FastAPI skeleton
- Baseline metrics utilities
- Active-learning utilities
- Dockerfile and Docker Compose
- GitHub Actions CI
- Issue templates
- Project charter, roadmap, benchmark design, and paper outline

## Scientific positioning

SensorGenome is not claiming to be the first fluorescence ML tool. The intended novelty is a condition-standardized benchmark for molecular sensor response, including analyte/interferent metadata, assay conditions, raw spectra, uncertainty-aware baselines, and active-learning selection of the next experiment.
