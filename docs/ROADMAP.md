# Roadmap

## Phase 0: Prior art and positioning
- Audit fluorescence datasets, molecular sensor databases, and AI probe-design tools.
- Write defensible novelty claim.
- Decide final name and repository scope.

## Phase 1: Schema and sample data
- Finalize probe-response schema.
- Define evidence grades.
- Add example data and validation CLI.

## Phase 2: Benchmark v0
- Add curated bimane/probe data.
- Implement scaffold splits.
- Add baseline models and uncertainty estimates.

## Phase 3: Active-learning simulator
- Candidate pool generator.
- Acquisition functions.
- Retrospective simulation.

## Phase 4: Prospective validation
- Select first candidate experiments.
- Add raw spectra upload format.
- Publish validation report.

## Phase 5: Community
- Documentation site.
- Contributor data templates.
- Dataset DOI.
- Benchmark leaderboard.

## Phase 6: Paper
- Submit benchmark/data paper.
