# SensorGenome Curation Rules

SensorGenome includes only experimentally reported molecular sensing systems.

## Required fields

Each entry should include:

- sensor name
- chemical structure or SMILES when available
- target analyte
- sensing modality
- response type
- solvent or medium
- pH when reported
- excitation wavelength when reported
- emission wavelength when reported
- detection limit when reported
- interferents tested when reported
- DOI or reliable literature source

## Do not include

- purely computational sensor proposals without experimental validation
- unclear chemical structures
- entries without a source
- unsupported claims of selectivity
- duplicated sensors unless experimental conditions differ

## Confidence score

Use:

- 3 = complete experimental data with structure, spectra, selectivity, DOI
- 2 = partial but reliable experimental data
- 1 = incomplete entry needing review
