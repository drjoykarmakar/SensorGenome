# SensorGenome-Bench v0.1 Milestone

## Goal

Release the first credible benchmark artifact for fluorescent small-molecule probe response.

## Deliverables

1. Probe-response schema v0.1
2. 25+ seed records
3. Raw spectra example format
4. Validation CLI
5. Baseline model CLI
6. Active-learning demo CLI
7. Benchmark metrics
8. Dataset card template
9. Model card template
10. v0.1 release notes

## Success criteria

- New user can install the package, validate the sample data, train a baseline, and run next-experiment selection in under 10 minutes.
- README clearly avoids exaggerated first-in-world claims.
- Repo has issues, docs, tests, and contribution pathways.
